# Changelog

## 1.0.0

- Initial release
- Status bar indicator showing the current Dial mode
- **Scroll** mode: scroll up/down, context menu on click
- **Zoom** mode: zoom in/out, reset zoom on click
- **Navigate** mode: navigate backward/forward in editor history
- **Debug** mode: start debugger / step into (click), step over / next breakpoint (rotate right), step out / previous breakpoint (rotate left)
- **Errors** mode: navigate to next/previous error, open Problems panel on click
- **Editor** mode: shift numbers and hex colors at the cursor position
- **Bookmarks** mode: toggle, navigate bookmarks (integrates with the Bookmarks extension if installed)
- **Find** mode: find next/previous match, open Find widget on click
- Configurable default mode and enabled mode list
- Configurable lines-to-scroll setting
